# TrainingFlutter
Flutter by Google

Sử dụng gói thư viện QR code
```
import 'package:qr_flutter/qr_flutter.dart';
```

Khởi tạo đối tượng với data là chuỗi string
```
    new QrImage(
            data: "M16225176",
            size: 200.0,
          ),
```
